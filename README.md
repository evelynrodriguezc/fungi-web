# Fungi Web

  A website about fungi built with plain HTML and CSS. No frameworks, no JavaScript - just getting comfortable with CSS the hard way.

  Practices modern CSS without shortcuts: custom properties, responsive layout with grid and flexbox, and consistent spacing through a design token system.

  ## Stack

  - HTML5
  - CSS3

  ## Run locally

  Open `index.html` directly in a browser, or:
